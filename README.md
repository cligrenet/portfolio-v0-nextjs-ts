# Portfolio

## About

This is my portfolio!

A [Next.js](https://nextjs.org/) project with TypeScript.

## Prerequisites

Node.js, Next.js, TypeScript

## Installation

```bash
npm install
# or
yarn install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Check port: [http://localhost:3000](http://localhost:3000)
